## 🎯 **DIRETRIZES DE USO**

### ✅ **PERMITIDO:**
- Protótipos de novas funcionalidades
- Dados de teste para desenvolvimento
- Exemplos de integração com APIs externas
- Mockups de telas e componentes
- Scripts experimentais

### ❌ **NÃO PERMITIDO:**
- Código de produção
- Dados sensíveis reais
- Credenciais ou tokens
- Arquivos grandes (>10MB)
